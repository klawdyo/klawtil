# klawtil

Util functions library

## Install

```sh
# Usando yarn
yarn add klawtil

## OU
# Usando npm
npm install klawtil

```

## Import

```js
// Modules
const { isCPF, isCNPJ } = require("klawtil");

// ES6
import { isCPF, isCNPJ } from "klawtil";
```
